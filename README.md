# Inventory CLI 

Sistem manajemen inventaris berbasis Command Line Interface (CLI) yang dibuat dengan Python. Program ini memungkinkan pengguna untuk mengelola data inventaris dan peminjaman barang. Program ini menggunakan file txt untuk menyimpan inputan dari user.

# Fitur

- **Inventaris**: Lihat, tambah, edit, dan hapus data barang.
- **Peminjaman**: Lihat, tambah, edit, dan hapus data peminjaman.

# Instalasi

1. Clone repository ini.
2. Pastikan Python 3.x terinstal.

# Penggunaan

Ikuti menu yang muncul di terminal untuk navigasi.
